# 🍽️ Exploratory Data Analysis on Zomato Dataset

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato restaurant dataset to uncover meaningful business insights related to restaurant ratings, cuisines, pricing, online delivery, and customer preferences.

The objective is to clean the data, visualize important patterns, and help businesses understand customer behavior for better decision-making.

---

## 📊 Dataset

- *Dataset:* Zomato Restaurants Dataset
- *Type:* CSV
- *Target:* Exploratory Data Analysis (No Target Variable)
